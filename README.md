# 障がい者就労継続支援B型 成長支援マネジメントシステム

利用者の成長を可視化し支援・業務・工賃に連動させるWebアプリ

## デプロイ

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/shuji0510/growth-support-system)

## ローカル実行

```bash
npm install
npm start
```

サーバーは http://localhost:3000 で起動します。

## 技術スタック

- Node.js + Express
- sql.js (SQLite in-memory)
- フロントエンド: HTML/CSS/JavaScript
